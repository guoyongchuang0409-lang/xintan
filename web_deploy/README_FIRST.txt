📁 文件说明
-----------
本目录包含心探应用的完整 Web 构建版本，可以直接部署到服务器。

主要文件:
  - index.html              应用主页面
  - main.dart.js           应用代码
  - flutter.js             Flutter 引导脚本
  - .htaccess              Apache 服务器配置
  - nginx.conf.example     Nginx 服务器配置示例
  - assets/                资源文件（图片、音效等）
  - canvaskit/             渲染引擎

配置文件:
  - DEPLOY_README.md       详细部署说明（必读！）
  - 部署检查清单.md         部署检查清单
  - quick_deploy.sh        快速部署脚本（Linux/Mac）

🚀 快速部署（3步）
------------------

方式一：使用静态托管服务（最简单）
  1. 注册 Vercel/Netlify 账号
  2. 拖拽上传本目录
  3. 完成！

方式二：使用自己的服务器
  1. 上传所有文件到服务器
  2. 配置 Web 服务器（参考 nginx.conf.example 或使用 .htaccess）
  3. 访问域名测试

方式三：使用快速部署脚本（Linux/Mac）
  1. chmod +x quick_deploy.sh
  2. ./quick_deploy.sh user@server:/path/to/deploy
  3. 完成！

📋 重要提示
-----------
1. 必须配置 URL 重写（用于 Flutter 路由）
2. 强烈建议启用 HTTPS
3. 建议启用 GZIP 压缩
4. 部署前请阅读 DEPLOY_README.md

⚠️ 常见问题
-----------
Q: 页面刷新后出现 404？
A: 需要配置 URL 重写，参考 .htaccess 或 nginx.conf.example

Q: 分享功能不工作？
A: Web Share API 需要 HTTPS，请配置 SSL 证书

Q: 如何配置 HTTPS？
A: 使用 Let's Encrypt 免费证书：
   sudo certbot --nginx -d your-domain.com
